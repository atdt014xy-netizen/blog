# 이미지 제작 가이드

## 대표 이미지 규격
- 가로 1080px, 세로 1080px (1:1 정사각형)
- 배경: 어두운 블루-퍼플 그라데이션 (#1a1a2e → #16213e)
- 메인 텍스트: 글 제목 (큰 글씨, 흰색)
- 보조 텍스트: 카테고리나 부제 (작은 글씨, 연한 회색)
- 모든 텍스트는 가로/세로 중앙 정렬
- 폰트: Pretendard 또는 시스템 기본 한글 폰트
- 우측 하단 또는 좌상단에 작은 액센트 (도형, 코드 심볼, 또는 단순한 아이콘)

## 본문 삽입 이미지 종류
- 비교 표 (제품/방법을 나란히 비교할 때)
- 단계별 다이어그램 (절차나 순서를 보여줄 때)
- 핵심 포인트 카드 (3~5개 요점을 정리할 때)
- 인용/강조 박스 (중요한 한 마디를 강조할 때)

## 공통 규칙
- 모두 HTML + CSS로 만들고, Python으로 Playwright를 실행해서 PNG로 캡처합니다.
- 배경 톤은 본문과 어울리게, AI/테크 분위기로 구성합니다.
- 텍스트는 최대한 짧게 유지합니다. 이미지는 보조 수단입니다.
- 코드 블록 스타일을 디자인 요소로 활용할 수 있습니다 (모노스페이스 폰트, 어두운 배경).
