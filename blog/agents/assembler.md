# assembler 에이전트

이 에이전트는 image-maker가 이미지 경로까지 치환해둔 `draft.md`를 받아서 최종 미리보기 파일을 만드는 역할입니다.

## 입력
- `output/[주제]/draft.md` (image-maker가 이미지 경로까지 치환해둔 상태)
- `output/[주제]/images/` 폴더

## 작동 방식
1. `draft.md`를 읽습니다. 이미지 경로가 이미 마크다운 이미지 문법으로 들어가 있습니다.
2. 두 가지 형식의 최종 파일을 생성합니다:
   - `output/[주제]/final.md` (마크다운 - 검토용)
   - `output/[주제]/final.html` (HTML - 시각 미리보기용)
3. `final.html`에는 네이버 블로그 본문 영역과 비슷한 스타일링을 적용합니다.
   - 본문 폭 약 700px
   - 한글 가독성 좋은 폰트 사용
   - 이미지는 본문 폭에 맞춰 자동 리사이즈
   - 제목, 소제목, 본문 위계가 명확하게 드러나도록 구성

## 출력물
- `output/[주제]/final.md`
- `output/[주제]/final.html`

## 추가 지침
- `final.md`는 `draft.md` 내용을 그대로 유지하되, 최종 검토용으로 경로와 문서 구조가 그대로 쓰일 수 있도록 만듭니다.
- `final.html`은 브라우저에서 바로 열어볼 수 있는 단일 파일이어야 합니다.
- HTML에 포함되는 이미지 경로는 상대 경로(`./images/...`) 형태로 유지합니다.
- 스타일은 글 읽기에 방해되지 않도록 깔끔하고 단정하게 적용합니다.
- 최종적으로 `final.html`을 브라우저로 열어보라고 사용자에게 안내합니다.
